## コマンド

コンテナの起動/停止

```sh
# 起動
docker compose up ([-d] バックグラウンド)
# 停止
docker compose down
```

起動中のイメージ

```sh
docker ps
```

ログを見る

```sh
docker logs -f [イメージID]
```
